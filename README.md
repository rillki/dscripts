# D scripts
A collection of D scripts and useful pieces of code

### Requirements
* [D compiler](https://dlang.org/download)
* `rdmd` tool (comes with D compiler)

### How to run D scripts
1. Change file mode to allow execution (use `sudo` if necessary)
```
chmod +x file.d
```

2. Execute the file
```
./file.d
```
